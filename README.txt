Project Overview:
This is a model of an Hospital Database Systems with 5 user roles, patients, doctors, hospital admins, community admins and system admins.
The following are the actions each role can perform.

New Patients can register themselves by creating a profile.
Existing Patients can search for doctors under the hospitals present under their community and book an appointment.
Doctors can enter VitalSigns report for a particular patient.
HospitalAdmins can manage the doctors under a given hospital.
CommunityAdmins can manage the hospitals under a given community.
SystemAdmins can perform CRUD on all the roles in the application.
